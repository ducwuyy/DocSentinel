## 📄 How to Use DocSentinel

DocSentinel works by checking documents for potential security and compliance issues. Here is how to use it:

1. Click **"Upload Document"** or drag files into the window. Acceptable file types include PDF, Word, Excel, and plain text files.
2. Select the file(s) you want to analyze.
3. Click the **"Start Assessment"** button.
4. DocSentinel will scan and parse your document.
5. The results page will show:
   - Identified risks
   - Compliance gaps
   - Suggested remediations
6. Use the report to understand issues and plan fixes.

You can save the report as a PDF or export it for sharing.
